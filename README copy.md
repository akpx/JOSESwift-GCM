# JOSESwift-AES256GCM

Modofied JOSESwift library for AES 256 GCM using CryptoSwift library 

System requirement
==================
Xcode - 11.3
Swift - 4.0

Library requirement
===================
CryptoSwift
CCCrypt <Still yet to test in JOSESwift>

Steps to Run
============
1. Open the project and run on simulator
2. Copy the generated framework to VMADemoiOS project
